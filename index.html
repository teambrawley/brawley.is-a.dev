Financejs Proxy Server

How To Use ?

Use it to quickly load any stock quote data from finance.js using a URL like :

https://finance.js.org/:stock_quote

To Load Net Share Purchase Activities :

https://finance.js.org/nspa/:stock_quote

To Load Financial Data :

https://finance.js.org/financial/:stock_quote

To Load Company Data :

https://finance.js.org/company/:stock_quote

User : ${user}

Voice : ${voice}
